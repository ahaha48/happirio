# Happirio 占いポータル — プロジェクト設計書

## 概要
「Happirio」は開運・占い系ブランドの無料占いポータルサイト。
GitHub Pages（happirio.github.io）でホスティング。サーバー代ゼロ。
全ツールは **単一HTMLファイル** で完結（CDN以外の外部依存なし）。

## 各ツールの仕様

### おみくじ（omikuji/）— 完成済み
- 7段階の運勢（大吉〜大凶）、確率重み付き
- ラッキーアイテム / ラッキーカラー / 今日のアクション
- Web Audio SE（振る音、紙をめくる音、鈴、大吉ファンファーレ、大凶SE）
- 1日1回制限（localStorage）
- SNSシェア（X / LINE）、#Happirio ハッシュタグ
- CTA: BOOTH開運待受 + note開運マガジン

### タロット（tarot/）— 既存あり・要カスタマイズ
- 大アルカナ22枚（正位置・逆位置）
- 2モード: 一枚引き（ファン選択UI）/ 二者択一（A/B比較）
- シャッフルアニメーション、カードフリップ、星空Canvas、BGM
- **要変更**: 1日1回制限を追加（各モード独立でOK）
- **要追加**: CTA セクション、SNSシェア、Happirio フッター
- **要検討**: 共通のナビゲーション（他の占いへの導線）

### ホロスコープ（horoscope/）— 新規作成
- 12星座の今日の運勢
- ユーザーが星座を選択 → 今日の運勢・ラッキーアイテム・ラッキーカラーを表示
- 運勢データは日付シードで決定論的に生成（同じ日・同じ星座なら同じ結果）
- AIのナレッジを活用し、各星座×運勢パターンの組み合わせを事前にJSON化
- 1日1回制限（選んだ星座を記憶、再訪時は結果を復元）
- デザインテーマ: 宇宙・星空系（タロットと近い世界観）
- CTA / シェア / フッターは共通仕様

### ポータルトップ（index.html）
- Happirio ロゴ / ブランド紹介
- 各占いツールへのカード型リンク（おみくじ / タロット / ホロスコープ）
- シンプルで美しいデザイン、和×宇宙のハイブリッドテーマ

## 技術的な注意事項
- 全ファイルは単一HTMLで完結させる（外部CSS/JSファイル分離は不要）
- GitHub Pages でホスティング（静的ファイルのみ）
- ネットワーク接続不要で動作する設計（API呼び出しなし）
- スマホファーストのレスポンシブ対応
- Web Audio API によるSE（ユーザー操作トリガー必須）
- localStorage で1日1回制限・結果の保存
